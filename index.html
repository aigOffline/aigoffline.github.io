Project 3
Predicting NYC Crime Rate Using Big Data Analytics Techniques

Data analyze and visualization of crime activities in NYC.

Technologies Used:

  • mySQL
  • Tableau Visualization
  • Altair visualization
  • Apache Spark
  • OpenRefine
  • QGIS

Project Difficulties

The problem of predicting violent crimes across NYC was particularly challenging. Demand is distributed in the city continuously and I needed Cluster. I issued with binary classification, data set is highly imbalanced.

My Solution

I group crime types into violent and nonviolent crimes and divided the data into a train and test set. Use a specific date such that before that day train data is around 70% and after that date test data is 30% of the all data set.

Libraries Used:
Technologies: NumPy, Pandas, Seaborn, Mathplotlib, Scikit-Learn
ML Algo: Unsupervised clustering

Project 4
Forecasting Hotel Demand Using Machine Learning Approaches

Hotel booking demand and cancellation rate forecast.

Technologies Used:

  • mySQL
  • PowerBI
  • RStudio

Project Difficulties

I had an issue splitting data into train and test to get the highest accuracy of the model.

My Solution

I use different methods, which is very important for weak prediction models.

Forecasting Methods:
Multiple logistic regression: 80.4% accuracy in predicting
Ridge regularization: 80.9% accuracy in predicting the model
Using Random Forest: 80% of the data as training data and 20% as testing I get 93.9% accuracy in predicting the model with testing dataset
Linear Regression: 90% of the data as training data and 10% as testing I get 73.0% accuracy in predicting the model
