Project 3

Data analyze and visualization of crime activities in NYC.

  • mySQL
  • Tableau Visualization
  • Altair visualization
  • Apache Spark
  • OpenRefine

Project Difficulties

The problem of predicting violent crimes across NYC was particularly challenging. Additionally, demand was distributed in the city continuously and I needed to use Cluster for it.

My Solution

I group crime types into violent and nonviolent crimes and divided the data into a train and test set. Use a specific date such that before that day train data is around 70% and after that date test data is 30% of the all data set.

Project 4
Forecasting Hotel Demand Using Machine Learning Approaches

Hotel booking demand and cancellation rate forecast.

Technologies Used:

  • mySQL
  • PowerBI
  • RStudio
  • Jupyter Notebook

Project Difficulties

Data set is highly imbalanced for many algorithms and I had an issue splitting data into train and test to get the highest accuracy of the model.

My Solution

I experiment with the different methods, which is very important for weak prediction models and used Agglomerative clustering.

Forecasting Methods:
Multiple logistic regression: 80.4% accuracy in predicting
Ridge regularization: 80.9% accuracy in predicting the model
Random Forest: 80% of the data as training data and 20% as testing I get 93.9% accuracy in predicting the model with testing dataset
Linear Regression: 90% of the data as training data and 10% as testing I get 73.0% accuracy in predicting the model
